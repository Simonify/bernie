bernie
===================

To install:
```
$ git clone git@github.com:Simonify/bernie.git
$ cd bernie
$ cp .env.sample .env
$ npm install
```

To start:
`$ npm run dev`.

To view:
[http://localhost:8356](http://localhost:8356)
